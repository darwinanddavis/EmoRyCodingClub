Before you arrive:

Please complete the following install & setup.

To participate in this workshop, you will need to make sure you have:

  • Installed R from CRAN.
    • If you’re asked to choose a CRAN mirror, just choose whichever is closest to your location.
  • Installed RStudio Desktop


Install Necessary Packages

Open RStudio and paste the following code into your console, then press Enter to run it:

# Download packages from CRAN

install.packages(c("devtools", "knitr", "magrittr", "sf", "sp", "rmarkdown", "usethis", "osmdata"))
